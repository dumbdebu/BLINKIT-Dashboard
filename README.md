Blinkit Sales Dashboard Analysis (Excel Project)

Overview

The current project represents an interactive Excel dashboard created to analyze Blinkit's sales performance, ratings, outlet trends, and products. Large-scale data sets were turned into actionable insights through data visualization, KPIs, pivot tables, and slicers.

The aim of the project was to gain experience with data analysis and dashboard creation in Excel along with gaining insights into how data analytics can be used by businesses to make their strategic decisions.


Features of the Project:

* Interactive Excel Dashboard
* KPI Cards Featuring Total Sales, Ratings, and Products Performance
* Pivot Tables and Pivot Charts
* Slicers Used to Filter Data Easily
* Analysis of Sales Based on Outlet Type and Location
* Product Category Performance Insights
* Outlet Establishment Trends
* User-Friendly Layout


Skills and Tools Applied:

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning
* Data Visualizing
* Dashboard Design
* Business Data Analysis


Information About the Dataset

The current dataset includes:

* Item Type
* Sales
* Outlet Type
* Outlet Size
* Outlet Location
* Item Ratings
* Fat Content
* Establishment Year

Key Learnings:

* Examined sales performances in terms of outlet type and size.
* Discerned which product segments produced better sales results.
* Compared performance metrics by location tiers of outlet.
* Analyzed customer ratings and trends in product visibility.
* Illustrated effect of years of outlet establishment on sales totals.
