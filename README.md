Blinkit Sales Dashboard Analysis (Excel Project)

Overview

This project is an interactive Excel dashboard created to analyze Blinkit sales performance, customer ratings, outlet trends, and product insights using raw retail data. The dashboard transforms large datasets into meaningful business insights through visualizations, KPIs, pivot tables, and slicers.

The objective of this project was to practice data analysis and dashboard creation in Microsoft Excel while understanding how businesses can use data to make strategic decisions.


Project Features:

* Interactive Excel Dashboard
* KPI Cards for Total Sales, Ratings, and Product Performance
* Pivot Tables & Pivot Charts
* Dynamic Slicers for Filtering Data
* Sales Analysis by Outlet Type and Location
* Product Category Performance Insights
* Outlet Establishment Trend Analysis
* Clean and User-Friendly Dashboard Design


Tools & Skills Used:

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning
* Data Visualization
* Dashboard Designing
* Business Data Analysis


Dataset Information

The dataset contains Blinkit retail sales data including:

* Item Type
* Sales
* Outlet Type
* Outlet Size
* Outlet Location
* Item Ratings
* Fat Content
* Establishment Year

The data was used to derive trends and insights related to product sales and outlet performance.


Key Insights:

* Analyzed sales performance across different outlet types and sizes.
* Identified which product categories generated higher sales.
* Compared outlet performance based on location tiers.
* Studied customer ratings and item visibility trends.
* Visualized how outlet establishment years impacted overall sales.

* Clean and organize raw data

* Use slicers for dynamic filtering

Although the project was inspired by a tutorial, I used it as a hands-on learning experience to strengthen my Excel and data analytics skills.
